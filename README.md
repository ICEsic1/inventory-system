# THE PRINTING DEPOT

A modern inventory management system built with Next.js, TypeScript, Tailwind, Supabase, and Recharts.

## Setup

1. Install dependencies: npm install
2. Configure Supabase env vars: NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY
3. Apply the SQL in supabase-schema.sql to your Supabase project.
4. Run: npm run dev
